Physical Tags
B
Bold
BIG
Increases the font size of the surrounding text
I
Italics
S
Strike Through - puts a line through text
SUB
Puts text below line - H2O
SUP
Puts text above line - X2
SMALL
Decrease font size of enclosed text
TT
Is viewed as type writer text
U
Underline
BR

Put thes the text that follows it on the next line or create a blank line
HR
Inserts a horizontal line
ADDRESS
Used to hold author's information and copyright
PRE
Allows you to put extra spaces that the browser will show. Do not use this tag with the P tag, but instead  of the P tag

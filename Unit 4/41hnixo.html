Logical Tags
  • To emphasize words of importance we can use the EM tag as many times as we need it. Sometimes we need a stronger feeling to an emphasized word and would be more apt to use the STRONG tag.
  • If we wanted to make some text stand out in a sentence so the reader would be aware of which words we wanted them to notice, we can use the CODE tag because it would make those words look differently than the rest of the sentence.
  • Finally, if you wanted to have a definition precede (come before) or follow a word, you could make this more obvious, easier to notice, by putting the DFN tags around the word to be defined.

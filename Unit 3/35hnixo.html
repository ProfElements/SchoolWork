Internet Vocabulary

This will show up fomatted as a paragraph. You will need to put a new paragraph tag before each paragraph.

WWW
short for World Wide Web
Web Browser
software used to access web pages.
Web Server
A computer used to host web content
URL
Uniform Resource Locator, called a web adress, reference to web content, specifying location on a computer network and a mechanism for retrieving it.
HyperText
text displayed on a computer with references to other text the reader can immediately access .
Cross-platform
computer software that is implemented on multiple computing platforms
Dynamic
collection of technology to create interactive and animated websites, using things like CSS, HTML, JS, and a DOM
Interactive
engangin and reacting to stuff done.
Web Page
document acting as a web resource
Web Site
collections of related network web resources, such as web pages and multimedia content
Home Page
Initial web page for a web browser
Menu structure
items listed in a ordered fashion to keep consistent across platforms
Linear Structure
collection of items ordered by single property so that each item has a unique predecessor except first and last.
HTML
Hyper Text Markup Language. The basic programming lang of web.
Links
A reference to to a thing or item.
Link Tag
A link between a document and an external resource, sometimes uses for stylesheets
Nested Tags
A collection of tags that inside each other being used for general organization
Remote Pages
A web page that is outside of the scope of related media you own.
Location Bar
Graphical display of the URL, users can easily type a new url to go to.
Link menu
A collection of links in an organized fashion
Anchor
A tag that is used to reference other material throughout the web

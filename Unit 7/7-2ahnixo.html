More Table Attributes

The following two tables use the ROWSPAN and COLSPAN attributes. When you want information to take up more space than one row you use ROWSPAN in the open TH or TD tag.

In the first example, I use ROWSPAN to create a heading that takes up two rows in one column. This means that when building my second row I only used two sets of TD tags for two columns because of the ROWSPAN.

In the second example, I use COLSPAN to create a heading that takes up three columns. This means that when building my first row I only use one set of TH or TD tags.

When you want extra space around the inside of a cell, use the CELLPADDING attribute in the open TABLE tag. When you want your table to have a colored border, use the BORDERCOLOR attribute in the open TABLE tag.

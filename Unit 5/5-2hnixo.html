Section 9.1 - Operations with Polynomials


Adding and Subracting Polynomials

Add the coefficients of like terms. Ex: 8x2 + 2x2 = 10x2

Subtract the coefficients of like terms. Ex: 8x2 - 2x2 = 6x2

Multiplying Polynomials

  1. monomial x monomial - coefficient x coefficient; variable x variable
    EX: (8xy2)(5x4y4) = 40x5y6
  2. monomial x polynomial - distribute the monomial to each term in the polynomial
    EX: 3x2y3 (2x2y + 3xy - 7) = 6x4y4 + 9x3y4 - 21x2y3
  3. binomial x binomial - FOIL
    EX: (x+7)(x+2) = x2 + 9x + 14
  4. binomial x trinomial - distribute each term in the binomial and add like terms
    EX: (x+2)(x2 + 3x + 4)
    • multiply first term : x3 +3x2 +4
    • multiply second term: 2x2 +6x + 8
    • add like terms : x3 + 5x2 10x + 8
  5. common patterns - You must memorize these patterns :
    • (a + b)(a - b) = a2 - b2 this is a difference of squares
    • (a + b)2 = a2 + 2ab + b2 this is a binomial squared
    • (a - b)2 = a2 - 2ab + b2 this is a binomial squared
    • (a + b)3 = a3 + 3a2b + 3ab2 + b3 this is a binomial cubed
    • (a + b)3 = a3 - 3a2b + 3ab2 - b3 this is a binomial cubed

Internet Vocabulary

This will show up fomatted as a paragraph. You will need to put a new paragraph tag before each paragraph.

WWW
an acronym for World Wide Web - a system of Internet Servers
Web Browser
software that is used to access web pages on the Internet
Web Server
a computer that delivers Web pages to your computer
URL
an acronym for Universal Resource Locator, a document's address on the Internet
HyperText
a special type of database system in which objects can be linked to each other
Cross-platform
the ability to run identically on more than one operating system
Dynamic
able to be changed as needed, Web pages are dynamic, book pages are not
Interactive
accepting input from a human
Web Page
a document on the World Wide Web
Web Site
a location on the World Wide Web that has Web pages, including a home page
Home Page
the main page of a Web site that usually contains an index to other documents
Menu structure
allows navigation of a Web site by choosing from a list of options
Linear Structure
allows navigation of a Web site in only one way, in a predetermined order
HTML
short for HyperText Markup Language, the authoring language for Web pages/dd>
